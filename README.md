# Alkemy Challenge Node.js
API para explorar el mundo de Disney, la cual permitirá conocer y modificar los
personajes que lo componen y entender en qué películas estos participaron.


## 🛠️Tecnologías usadas:
* Node.js
* Express
* JWT
* JOI
* Sequelize
